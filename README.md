## Featured Projects

### HazardNet – Disaster Mapping and Prediction System

HazardNet is a research-based disaster alert and hazard prediction system for Sri Lanka. It uses machine learning to predict flood and landslide risks and provides users with real-time predicted data and safety alerts.

**Technologies:** Python, Machine Learning, Spring Boot, Flutter, MySQL, APIs

### Craft IQ

CraftIQ is a full-stack skill-sharing and learning platform where users can create posts, manage learning plans, interact with others, and track learning progress.

**Technologies:** React.js, Spring Boot

### UniTime

UniTime is an academic scheduling platform that supports professor selection, room allocation, and time slot recommendations using machine learning concepts.

**Technologies:** React.js, Spring Boot, Python

### ChocoCart

ChocoCart is a chocolate shop web application where admins can manage products and customers can add products to their cart.

**Technologies:** Node.js, React.js

### TaskManager

TaskManager is a mobile Todo application developed with Kotlin and SQLite. It allows users to add, update, delete, and prioritize tasks.

**Technologies:** Kotlin, SQLite

### Solifex Private Limited Website

A real-world business website developed for Solifex Pvt Ltd to showcase solar energy products and services. I also handled hosting, domain configuration, and server setup.

**Technologies:** HTML, CSS, JavaScript, PHP

### TaskMaster

TaskMaster is a job posting, bidding, and eCommerce web application. My contribution focused on the eCommerce store and admin product management features.

**Technologies:** Angular, Spring Boot, MySQL
